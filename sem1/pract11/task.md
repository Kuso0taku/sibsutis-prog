# Task  
Create a structfunc.h and structfunc.c file to implement
The structure and functions for processing it are described below.
The `main.c`, contains the `int main` function
and includes the `structfunc.h` header file.
**Streams** were also used for input and output.

Example of running the funcion: `./main < in > out`  

### Sub-tasks

1. Create an array of structures consisting of 10 data objects. 
2. For the selected structure, implement a function for entering structure elements (using a **pointer** to the structure). (setter)
3. For the selected structure, implement a function to output the values of the specified structure (using **structure** as a value, **protect from modification**) (view)
4. Create a file for `<` inputting data for 10 structures.
5. The result of the program is the output of 10 entered values to the file via `>`.

### Var 7
- Creating a structure for storing data about computer games (genre, developer, platform, release date).
