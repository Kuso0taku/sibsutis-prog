# Task  
Create `stringfunc.h` and `stringfunc.c`  
where to implement the text processing functions described below.  
When implementing these functions,  
functions from the `string.h` library **MUST NOT be used**.  
The main file, `main.c`, contains the `int main` function   
and includes the `stringfunc.h` header file.  
When launched, the program is passed **arguments** to  
the **function number** to be executed.  
**Streams** `<` and `>` are also used for *input* and *output*.  
  
Example of running the funcion: `./main 2 < in > out`  
  
There's no need to display the *menu*,  
but you should use **comments** in the code to indicate the function assignment.
